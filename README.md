# 🎬 Movies App

A simple movie web app where users can browse and watch uploaded movies.

## 🔥 Features

* 🔍 Search movies by name
* 🎥 Upload your own movies
* ⭐ Show movie rating
* 🖼️ Display movie posters
* 📱 Responsive design

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 📂 Project Structure

```
├── index.html
├── style.css
├── script.js
```

## 🚀 How to Run

1. Download or clone this repo
2. Open `index.html` in your browser
